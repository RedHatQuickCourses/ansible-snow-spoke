= Operational Flow: From Trigger to Execution

The flow doesn't just live inside ServiceNow. It can reach out via REST API to the Ansible Automation Platform (AAP) to launch playbooks, effectively bridging the gap between an ITSM request and infrastructure-level execution. This orchestration ensures that business logic is strictly enforced while providing a transparent, auditable trail of every automated step taken across the enterprise.

. Navigate to *Process Automation* → *Flow Designer* in ServiceNow, which opens a new tab. 
+
In the newly opened tab, click *Actions*, search for “Job" , and select the corresponding action:

Congratulations! After completing these steps, you can successfully use a ServiceNow Catalog Item to launch job templates in Ansible Automation Platform via the Ansible Spoke. This integration enables end users to leverage a familiar interface while triggering powerful backend automation, significantly improving efficiency and reducing time to value across the organization.
